\# Response Playbooks



\### Critical / High Severity

1\. Isolate host (if server or suspicious process)

2\. Capture memory dump

3\. Full process tree + network connections

4\. Check lateral movement indicators

5\. Escalate to Incident Response



\### Medium Severity

1\. Contact user within 1 hour

2\. Review downloaded/uploaded files

3\. Document justification

4\. Monitor for recurrence



\### Low / Informational

1\. Log event

2\. Optional user awareness email

3\. Track frequency (escalate if repeated)
